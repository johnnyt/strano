Strano is a set of tools to manage "SliceHost":http://www.slicehost.com/ slices - using Debian 5.0 (lenny) - and Ruby on Rails applications using:

* [[Slice Capistrano Tasks]] - Currently for "SliceHost":http://www.slicehost.com/ slices
* [[Application Rake Tasks]]
* [[Application Capistrano Tasks]]
* Central AES encrypted [[Variables]] YAML File which contains all passwords and any other company secure information - the master password must be entered in order to execute any Capistrano task.

h2. Examples

Server Setup:
<code>cap slice:setup</code>
